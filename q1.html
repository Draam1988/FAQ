Инструкция по установке Happ

Сейчас мы рекомендуем всем нашим клиентам использовать приложение Happ.

Happ — это кроссплатформенное приложение для удобной работы с прокси-серверами на базе мощного ядра Xray. Оно доступно на Windows, macOS, Linux и мобильных устройствах (iOS/Android).

Интуитивный интерфейс и гибкие настройки делают Happ надёжным инструментом. Вам не нужно думать, как настраивать VPN на разных устройствах — схожий и дружелюбный интерфейс делает установку очень простой.

1.Скачайте и установите Happ

Apple (iPhone, iPad, MacBook):
• Для RU App Store
• Глобальная версия

Android:
• Google Play Store

Десктоп версии:
• Windows (.exe)
• Linux (.deb)

Если нет учетной записи Google/Apple, скачайте файлы напрямую:
• APK для Android
• DMG для macOS

2.Скопируйте ключ подписки

Скопируйте текст/ссылку, которую выдал вам бот при покупке.

ВАЖНО! Копируйте ключ целиком! Если в ключе есть "флаг/смайлик и русский текст" — они обязательны для корректной работы.

3.Ввод ключа в приложение

В верхнем правом углу Happ нажмите на кнопку + (плюс) и выберите пункт: "Вставить из буфера обмена". После этого нажмите на большую круглую кнопку для запуска.

4.Несколько ключей

Вы можете добавить несколько ключей. Чтобы переключиться, просто нажмите на нужный ключ — он выделится цветом.

5.Проверка соединения

Нажмите на значок спидометра или надпись "проверить текущее соединение", чтобы убедиться, что всё работает.
